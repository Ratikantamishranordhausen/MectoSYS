## Product Scope (Frontend)
The frontend delivers a maintenance and operations interface for industrial workflows. It includes dashboards, work-order management, calendar planning, and interactive machine views built with a consistent branding system.

## Core Features
- Maintenance calendar (month/week/day views) with rich filtering, search, and keyboard shortcuts.
- Work order creation (TBM/CBM) with validation and detailed execution tracking.
- Drag-and-drop rescheduling and responsive mobile layout.
- Visual status badges, priority color coding, and overdue indicators.
- SOP-linked procedures and detailed modal-based inspection flows.
- Print/PDF-ready layouts for reporting and audits.

## Key Screens (Representative)
- Dashboard and KPI summaries
- Calendar and scheduling views
- Work order create/execute flows
- Asset/machine pages
- Inventory and audit views
- Public web pages (marketing and onboarding)

## Frontend Modules (High-Level)
- Templates: HTML views for app and web pages
- Static assets: CSS, JS, and images
- Shared layout: navigation, sidebar, and footer components

## Tech Stack (Frontend)
- HTML templates
- CSS (Tailwind via CDN for specific modules)
- JavaScript (client-side interactions)
- Font Awesome and Google Fonts

## Design & UX Principles
- Clean, professional UI with consistent branding
- Mobile-first responsiveness and touch-friendly controls
- Accessible forms and clear validation states
- Progressive enhancement with graceful fallbacks
